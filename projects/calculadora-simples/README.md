# Calculadora Simples

Projeto desenvolvido durante meus estudos de Python.

## Objetivo

Criar uma calculadora capaz de realizar as quatro operações matemáticas básicas:

* Soma
* Subtração
* Multiplicação
* Divisão

O programa permanece em execução até que o usuário escolha encerrar.

## Conceitos praticados

* Funções
* Parâmetros e retorno (`return`)
* Estruturas condicionais (`if`, `elif`, `else`)
* Laços de repetição (`while`)
* Entrada de dados com `input`
* Validação de entrada do usuário
* Tratamento simples para divisão por zero

## Tecnologias

* Python 3

## Como executar

Execute o arquivo:

```bash
python main.py
```

Escolha uma das operações disponíveis:

```text
soma
subtracao
multiplicacao
divisao
```

Digite os números desejados e o programa exibirá o resultado.

Para encerrar a calculadora, digite:

```text
0
```

quando for solicitado o nome da operação.

## Exemplo de uso

```text
Digite a operação que quer realizar: soma/subtracao/multiplicacao/divisao. Ou digite 0 para encerrar.

soma

Digite o primeiro número: 10
Digite o segundo número: 5

15
```
