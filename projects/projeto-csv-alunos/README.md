# Média de Alunos com CSV

Projeto simples desenvolvido durante meus estudos de Python.

## Objetivo

Cadastrar alunos e suas notas em um arquivo CSV e depois:

- Calcular a média da turma
- Encontrar a maior nota
- Encontrar a menor nota
- Identificar os alunos correspondentes

## Conceitos praticados

- Listas
- Tuplas
- Laços de repetição
- Condicionais
- Leitura de arquivos
- Escrita de arquivos CSV
- Processamento de dados

## Tecnologias

- Python 3

## Como executar

Execute o arquivo:

```bash
python main.py
```

Digite os alunos e suas notas.

Para encerrar o cadastro, digite:

```text
0
```

no campo nome.
